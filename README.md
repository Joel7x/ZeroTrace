# ZeroTrace

Healthcare #5: Claims Fraud Detection with Risk Scoring

---

## Problem Statement

Healthcare claims get submitted, paid, and audited with no one catching patterns that suggest fraud. Build a system that analyzes claims, detects suspicious patterns, assigns risk scores, and provides traceable reasoning for every flag it raises.

---

## Features

- Fraud risk scoring
- Claim analysis
- Explainable fraud reasons
- Audit history tracking
- Interactive dashboard
- Search and filtering
- Data visualization charts
